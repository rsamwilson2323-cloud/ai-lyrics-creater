# 🎵 AI Lyrics Creator

AI Lyrics Creator is a web-based application that generates **creative song lyrics using Artificial Intelligence**.  
Users can enter a **theme, mood, or idea**, and the AI will generate unique lyrics instantly.

This project demonstrates how AI APIs can be integrated with **Python and a simple web interface** to create creative musical content.

---

# 🚀 Features

✨ Generate song lyrics using AI  
🎶 Enter your own **theme or topic**  
⚡ Fast lyric generation  
🌐 Simple and clean web interface  
🧠 Powered by AI language models  
💡 Useful for musicians, writers, and creators  

---

# 🧠 How It Works

1. User enters a **theme or idea**
2. The application sends the prompt to the **AI API**
3. The AI generates **creative lyrics**
4. The lyrics are displayed instantly on the webpage

# ⚙️ Requirements

Before running this project, make sure you have the following installed:

- Python **3.8 or higher**
- pip (Python package manager)
- Internet connection

---

# 📦 Python Libraries

Install required libraries:

```
pip install groq
pip install flask
pip install python-dotenv
```

Or install everything at once:

```
pip install -r requirements.txt
```

---

# 🔑 API Setup

This project requires a **Groq API Key**.

1. Create an account at:
```
https://console.groq.com/
```

2. Generate your API Key.

3. Add the key in `app.py`:

```python
GROQ_API_KEY = "YOUR_API_KEY"
```

---

# 💻 Installation

Clone the repository:

```
git clone https://github.com/rsamwilson2323-cloud/ai-lyrics-creater.git
```

Go to the project folder:

```
cd ai-lyrics-creater
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the application:

```
python app.py
```

---

# ▶️ Usage

1. Start the application
2. Open the browser
3. Enter a **song topic or mood**
4. Click **Generate Lyrics**
5. The AI will create lyrics instantly

Example prompt:

```
Love breakup song
```

Example output:

```
Verse:
In the quiet night I hear your name,
Echoes dancing through the rain...
```
